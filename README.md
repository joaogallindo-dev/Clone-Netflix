🎬 Clone-Netflix
---

O projeto é um clone da Netflix desenvolvido para fins de estudo, com foco na integração de uma API. Sua estrutura visual e funcional é inspirada na plataforma original, permitindo explorar recursos modernos de interface e consumo de dados.

<br>

📄 Funcionalidades
--- 
- Reponsividade
- Navegação intuitiva  
- Integração com TMDb API
